Serie de 8 estudios de campo sobre CSS nativo, accesibilidad (APCA/OKLCH) y eliminación de deuda técnica (Baseline 2026)
# Ignacio Llorente Fdez de la Reguera
### Diseñador Gráfico & Design Technologist

Llevo desde 1999 uniendo la estética del diseño gráfico con la precisión técnica de los estándares web. Mi enfoque profesional se centra en la construcción de interfaces mediante código nativo, priorizando la accesibilidad universal y la eficiencia técnica.

---

**Áreas de especialización:**

 * Consultoría en Estándares W3C y Accesibilidad (WCAG / APCA).
 * Arquitectura de interfaces con CSS moderno (OKLCH, Container Queries).
 * Diseño y desarrollo de sistemas de diseño (Design Systems) sin dependencias de frameworks.

---

**Filosofía técnica:**

Defiendo una web basada en "0 Fósiles": menos abstracción, menos deuda técnica y un compromiso ético con el/la usuario/a final. El estándar no es solo una norma, es el camino hacia una web más humana y sostenible.

Menos código + Más rendimiento = Más inclusión (A11Y) | Propiedades Nativas = Navegador optimizado = 60 fps constantes

"El estándar es el camino" 🥷
